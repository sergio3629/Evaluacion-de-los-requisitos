DEFINICIÓN DEL PROYECTO




Este proyecto se basa en crear un aplicativo el cual se pueda realizar compras virtualmente, para poder mejorar el tema de ventas dentro de los centros comerciales de Pitalito, evitar la necesidad de transportarse hacia los centros comerciales para realizar las compras precensialmente y evitar posibles riesgos.

  • Elementos necesariso para ejecutar la idea de negocio:

    • Tener los suficientes recursos para la ejecución del negocio y tener conocimientos sobre el tema de comercio dentro de los centros comerciales y los usuarios.

    - Hadware:

    Se necesitaria dos equipos de computo, con buen almacenamiento, procesador y accesoa internet.

    - Software:

    Se necesitaria acceso a internet, un minimo de 2GB de RAM, un minimo de 30GB de almacenamiento, porgrmas y herramientas para diseñar y crear el aplicativo.


  • Presupuesto estimado para la elaboración del negocio:

    • Para la elaboracion del negocio se necesitaria 30.000.000$.


  • Licensias que se requieren:

    • Se necesitaria licencia de dominio, licencia de bases de datos en la nube, permisos en la Play Store y App Store.
